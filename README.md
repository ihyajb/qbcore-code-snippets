## Installation

1. Install Visual Studio Code 1.58.0 or higher
2. Launch Code
3. On the left side of Visual Studio Code select `Extensions`
4. Click "Search Extensions in Marketplace"
5. Type `QBCore`
6. Click Install
7. Reload Visual Studio Code
